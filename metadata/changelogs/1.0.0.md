# 1.0.0

Initial release.

* Mixin-patch `MachineRoomScreen.updateScene` to dispose the prior
  `BakedLevel` before the renderer is replaced.
* Mixin-patch `MachineRoomScreen.onClose` to dispose the active
  `BakedLevel` when the preview screen closes.
* All `VertexBuffer.close()` / `SectionBufferBuilderPack.close()` calls
  are routed onto the render thread via `RenderSystem.recordRenderCall`.
* Gander internals are accessed reflectively via record-accessor names
  so the patch is forward-compatible with field-type changes.
* `@Pseudo` mixin and `require = 0` injects mean the mod is a silent
  no-op when Compact Machines is absent or when CM's class layout
  changes incompatibly.
